# Student Budget Tracker

## Overview
Student Budget Tracker is a Java console-based application developed to help students manage their daily expenses and monthly budget effectively. The project is designed especially for hostel and college students who often face difficulty in tracking their spending habits.

This application allows users to add expenses, view all recorded expenses, calculate the total amount spent, set a monthly budget, check remaining budget, search expenses by category, delete expenses, and save/load expense data using file handling.

---

## Problem Statement
Many students, especially hostellers, spend money daily on food, travel, study materials, and personal needs but often do not maintain a proper record of these expenses. This can lead to overspending and poor budget management.

The purpose of this project is to provide a simple and user-friendly solution for tracking expenses and managing monthly budgets using Java programming concepts.

---

## Features
- Add a new expense with date, category, amount, and description
- View all recorded expenses
- Calculate total amount spent
- Set monthly budget
- View remaining budget and budget status
- Search expenses by category
- Delete an expense
- Save expenses to a file
- Load saved expenses automatically when the program starts
